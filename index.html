Oftewel; wat heb je nodig? en hoe te beginnen?

JavaScript

JavaScript, in welke smaak je maar wilt:
Vanilla, of vanuit een framework: React, Vue, Angular, Svelte, etc...

API's

API's, Verbindalles met elkaar...
Maak applicaties krachtig door gebruik te maken van API's.

Markup

Markup, zoals de M uit HTML...
De templates van de applicatie: in HTML, Markdown, Pug, of welke template language je maar wilt

Maar wanneer is het nu een Jamstack applicatie?

Als:

  • ... de statische templates geladen worden van een CDN
  • ... de content dynamisch wordt ingeladen vanuit een API
  • ... er gebruik wordt gemaakt van de JavaScript voor de logica

Hoe maak ik een Jamstack applicatie met JavaScript?

Door gebruik te maken van Static Site generators als:

  • Gatsby
  • NextJs
  • Eleventy

Services

  • API's voor content van derde partijen
